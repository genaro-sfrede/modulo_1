Breve Descripción

Este es un párrafo desarrollado integramente con el procesador de texto integrado en GitHub al cual accedimos mediante la utilización del comando Ctrl + ">". Este procesador de texto es una version compacta del editor de texto Visual Studio Code, de propiedad de Microsoft.

Además, recurrimos tambien a la creación de un archivo cuyo nombre es "INDEX.html" que contiene precisamente este código.
